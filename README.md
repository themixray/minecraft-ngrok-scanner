# Minecraft Ngrok Scanner
Scans ngrok tcp addresses on running Minecraft servers

**Requires [mcstatus](https://github.com/Dinnerbone/mcstatus)**

## Changeable constants
```py
countries = ['','.au','.eu','.ru']
filterVersion = None
minimumPlayers = 1
logpath = f"C:\\Users\\{getpass.getuser()}\\servers.log"
```
